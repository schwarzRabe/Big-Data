Описание
Проект по анализу метаданных рентгеновских снимков для эпидемиологического мониторинга COVID-19 с использованием PySpark.
Датасет: COVID-19 Chest X-Ray Dataset
Основные этапы:

Предобработка и очистка данных
SQL-аналитика (5 запросов)
Обработка в PySpark (UDF, фильтрация)
Визуализация (4 графика)
